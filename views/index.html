API Rest de presidentes y operadores políticos de Argentina

Para ver todos los presidentes hay que hacer un GET a http://localhost:3000/presidents

Para traer un presidente por ID, hay que hacer un GET a http://localhost:3000/presidents pasándole como parámetro su ID

Para ver todos los presidentes filtrados por edad y traer a los mayores de esa edad en cuestión, hay que hacer un GET a http://localhost:3000/presidents/age , pasándole como query la edad

Para ver todos los presidentes filtrados por ideología, hay que hacer un GET a http://localhost:3000/presidents/ideology pasándole como query la ideología. "Centrista", "Derecha" o "Izquierda"

Para ver todos los presidentes de forma paginada hay que hacer un GET a http://localhost:3000/presidents/paginated/ indicandole por query el número de página(paginated) y la cantidad de presidentes(limit) a traer por cada página

Para traer a los presidentes ordenados por su edad de manera ascendente, hay que hacer un GET a http://localhost:3000/presidents/age/order

Para crear un presidente, es necesario hacer un POST a http://localhost:3000/presidents indicando "name", "age", "description", "ideology", "status", donde estatus es si está vivo (como true) o muerto (como false).

Para eliminar un presidente, es necesario hacer un DELETE http://localhost:3000/presidents pasándole como parámetro el ID

Para editar un presidente, es necesario hacer un PUT a http://localhost:3000/presidents pasándole como parámetro el ID

Crear una cuenta

Para crease una cuenta, hay que hacer un POST indicando el "name", "email" y "password" a http://localhost:3000/fixers. Para obtener un token y poder hacer las solicitudes de presidente, hay que hacer un POST a http://localhost:3000/auth con el usuario y la contraseña Recorda que tu nombre debe tener mas de dos caracteres y no contener caracteres extraños ni espacios.
